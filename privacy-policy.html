隐私政策

更新日期：2026-06-02
生效日期：2026-06-02

本应用《食愈》（以下简称“本应用”）由个人开发者提供。你在使用本应用前，请仔细阅读本政策。本政策说明当前版本中与个人信息、设备权限、第三方服务相关的处理方式。

一、我们处理的信息类型

  • 饮食记录信息：你主动录入或确认保存的食物名称、原始输入、克重、餐次、日期/时间、营养估算、抗炎/促炎分析结果、记录来源、模型解析结果等。
  • 智能识餐配置：你在应用内填写的 OpenAI 兼容接口地址、API Key、模型名称、详细营养开关、日切设置等。
  • 语音输入信息：你主动长按语音输入时产生的临时录音文件，以及端侧转写得到的文字。
  • 本地偏好与设置：昵称、主题、提醒开关和提醒时间、协议同意状态、开发者模式等。
  • 导入导出信息：你主动导入的备份 CSV 文件内容，以及主动导出时生成的 CSV 文件。
  • 操作日志信息：应用会在本地记录少量操作日志，用于排查导入导出、智能识餐、提醒、语音识别等问题。

二、信息处理目的

  • 提供饮食记录、餐次详情、每日详情、历史统计、洞见趋势、抗炎/促炎分析等核心功能。
  • 将你输入的自然语言饮食描述解析为结构化饮食记录。
  • 在你主动开启时提供本地提醒、语音输入、导入导出、反馈排查等功能。
  • 检查协议版本、展示隐私政策/用户协议和应用更新入口。

三、处理方式与范围

  • 饮食记录、设置、协议同意状态、日志等默认保存在你的设备本地，主要使用 SharedPreferences、应用支持目录或系统临时目录。
  • 本应用当前不提供开发者服务器账号体系，也不主动将你的本地饮食记录同步至开发者服务器。
  • 使用智能识餐时，你输入的饮食描述、相关当日上下文和必要的提示词会发送至你自行配置的 OpenAI 兼容 API 服务商；API Key 仅保存在本机，并仅作为请求授权信息发送给你配置的服务商。
  • 语音输入目前在 Android 端使用本地语音识别。首次使用或模型缺失时，应用可能从公开模型地址下载离线识别模型到应用支持目录；录音文件保存在系统临时目录，转写完成、取消或失败后会尽量删除，不会为了语音识别上传录音。
  • 当你主动导出并分享数据时，应用会生成 CSV 文件并调用系统分享面板；文件可能被发送至你选择的第三方应用。
  • 当你主动导入数据时，应用仅读取你通过系统文件选择器选中的 CSV 文件。
  • 当你主动复制诊断日志或反馈模板时，应用会写入系统剪贴板；不会后台读取剪贴板内容。

四、第三方 SDK、库与服务

  • shared_preferences：本地保存饮食记录、设置、协议状态和日志。
  • http：请求你配置的智能识餐 API、远程协议文本、应用更新地址，以及下载本地语音识别模型。
  • webview_flutter：在应用内展示隐私政策、用户协议等网页。
  • url_launcher：打开外部链接或系统邮件应用。
  • file_selector：调用系统文件选择器用于导入备份。
  • share_plus：调用系统分享能力用于导出备份。
  • flutter_local_notifications、timezone：调度本地饮食提醒；不使用第三方推送服务器。
  • record：在你主动长按语音输入时录制临时音频。
  • sherpa_onnx、archive、path_provider：下载、解压并运行端侧离线语音识别模型。
  • fl_chart、flutter_svg、package_info_plus：用于本地图表、图标展示和版本信息读取。

本应用当前未集成广告、统计归因、社交登录、支付类 SDK。智能识餐 API 服务商由你自行配置，其数据处理规则请以对应服务商的隐私政策为准。

五、权限与设备能力

  • 网络访问：用于加载协议/更新链接、调用你配置的智能识餐 API、下载端侧语音识别模型等。
  • 麦克风权限：仅在你主动使用语音输入时请求，用于录制临时音频并在本机转写。
  • 通知权限：仅在你开启提醒时用于展示本地饮食提醒。
  • 开机完成广播：用于在设备重启后恢复已开启的本地提醒。
  • 文件选择与读取：仅在你主动导入备份时触发。
  • 系统分享：仅在你主动导出备份时触发。
  • 剪贴板写入：仅在你主动复制日志或反馈模板时触发。

六、你的权利

  • 你可在应用“我的”页面查看隐私政策、用户协议、开源许可和版本信息。
  • 你可在应用内修改或删除饮食记录，并可通过“清空本地数据”删除本地饮食记录和相关评分缓存。
  • 你可在设置中修改昵称、主题、提醒、智能识餐 API 配置等信息；也可清空诊断日志。
  • 你可停止使用本应用来终止后续信息处理。卸载应用通常会删除应用本地数据，具体以设备系统机制为准。

七、未成年人保护

若你为未成年人，请在监护人指导下阅读并使用本应用。本应用提供的饮食分析仅用于一般健康管理参考。

八、政策更新

我们可能根据功能与法规变化更新本政策。更新后将通过应用内版本信息或重新同意流程进行提示。
