用户协议

更新日期：2026-06-02

欢迎使用《禾宿》应用。你使用本应用即视为已阅读并同意本协议。如你不同意本协议，请停止使用本应用。

一、服务内容

  • 本应用提供饮食记录、智能识餐、餐次详情、每日详情、历史统计、洞见趋势、抗炎/促炎分析、本地提醒、语音输入、导入导出等功能。
  • 智能识餐会根据你输入的文字或语音转写文本，生成待确认的饮食记录；你确认后，记录才会保存到本地。
  • 本应用内容用于一般健康管理参考，不构成医疗诊断、治疗建议、处方、营养治疗方案或其他专业医疗服务。

二、用户行为规范

  • 你应确保录入、导入、导出和分享的数据不违反法律法规及他人权益。
  • 你应自行核对智能识餐、营养估算、抗炎/促炎分析等结果。对于识别错误、估算偏差或记录遗漏，你可在应用内修改后再保存或使用。
  • 你应妥善保管通过导出功能生成的数据文件，以及自行配置的 API Key 等敏感信息，避免泄露。
  • 你不得利用本应用从事违法违规活动，或以破坏、干扰、逆向攻击等方式影响应用正常运行。

三、智能识餐与第三方服务

  • 如你配置并使用 OpenAI 兼容接口，饮食描述和相关上下文会发送至你选择的 API 服务商处理。
  • 你应自行确认所使用 API 服务商的服务条款、隐私政策、计费规则和可用性。本应用不对第三方 API 服务商的响应速度、识别准确性、费用、故障或数据处理行为承担超出法律规定范围的责任。
  • 语音输入目前主要依赖 Android 端侧语音识别模型；不同设备、系统版本、网络环境和模型状态可能影响可用性。
